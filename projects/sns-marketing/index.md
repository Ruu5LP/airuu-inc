# SNSマーケティング

AiRuu Inc.（Ruuの人生を会社として経営する、というコンセプト）と、Ruuの開発者としてのブランディングを
X（旧Twitter）と note で発信していくための活動をまとめる場所。継続的な取り組みなので、1ファイルに詰め込まず
**フォルダ構成**にして、増える調査・下書き・投稿ログをファイル単位で足していく。

## 目的

Ruuが寝ている間にAiが調査・下書きまでやっておき、朝起きたら選ぶだけの状態にする。実際に投稿したら[[投稿ログ]]に記録し、
反応を見ながら型を育てていく。

## 状態

**記事1「人生を"会社"として経営してみることにした話」を公開済み**（無料公開）。詳細は[[2026-07-22-first-note-published]]を参照。

## 次の一歩

- Xの告知ポスト（記事1へのリンク付き）を出すタイミングを決める。調査結果通り木曜夜が狙い目
- 1本出してみて反応を見てから、[[振り返り]] に「発信の手応え」を記録する
- 記事2・3も含め、良かった/悪かったを[[投稿ログ]]に追記して、型を育てていく

## Ruuへの持ち帰り事項（Aiが判断を保留したこと）

作業を止めずに進めるため、Ruu自身の判断が要る点はここに溜めておく。

- **仕事の納期調整ネタ**：意思決定の型を使った実例として強いネタだが、職場が特定されうるリスクがあるため下書きは抽象化してある（[[日記からのネタ棚卸し]]参照）。実際に出す前に、Ruu自身の目で「これなら大丈夫」と確認してから使ってほしい。
- **マーケティング系プラグイン（`marketing`）の導入見送り**：Ahrefs/HubSpot/Amplitudeなど未接続の外部ツール前提のスキル束で、今の環境では実質使えないと判断してインストールしなかった。将来それらのツールと連携するなら改めて検討の余地あり。
