# 投稿ログ

実際にX/noteに投稿したものと、その反応・手応えを記録する場所。新しいものを上に追記していく。

## 書き方

- 日付 / プラットフォーム / タイトル・内容の要約 / リンク / 反応メモ、を1エントリにする
- 数値（いいね・リポスト・スキ数など）は分かる範囲でOK。厳密さより「続ける」を優先
