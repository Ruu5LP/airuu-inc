# プロジェクト

進行中の取り組みを置く場所。開発・学び・健康・お金――「会社」として動いているものを、
1プロジェクト1ノートで管理する。止まっているものに気づくための一覧でもある。

## 書き方

- 1プロジェクト = 1ノート。ファイル名は ASCII（英数字・ハイフン）で（例: `airuu-repo-setup.md`）。表示名は frontmatter の `title` に日本語で書く。
- 各ノートに入れたいこと:
  - **目的** — 何のためにやるか。
  - **状態** — 進行中 / 保留 / 完了。
  - **次の一歩** — 今いちばん小さく動ける行動。
  - 関連する [[決定ログ]] や [[振り返り]] に wikilink で繋ぐ。

## 一覧

- [[SNSマーケティング]] — 進行中（記事1公開済み）
- [[麻雀トラッカー]] — v1完成・稼働中（家麻雀のスコア＋局データ記録ツール）
- [[ループエンジニアリング]] — 学習中・設計フェーズ（AIエージェントを自走させるループの設計手法）
